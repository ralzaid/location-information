# Assignment 4 - Location Information

## Overview

This project creates a location-based app using Google Maps in Jetpack Compose.

The app requests location permission and displays a map centered on the user’s current location. It shows a marker for the user and displays the address at the top of the screen. Users can also tap on the map to add custom markers.

## AI Usage Disclosure

I used ChatGPT to help debug location issues and understand how to correctly implement the Google API & retrieve the user’s current location.
